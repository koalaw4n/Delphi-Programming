Build Configuration

Sebelum mendistribusikan aplikasi, ubah konfigurasi dari Debug ke Release.

Mode Release

Menghapus simbol debug, memperkecil ukuran file EXE, dan mengoptimalkan performa kode.

Project Manager

Klik kanan pada Build Configurations > Release > Activate, lalu klik Build.

Dependency & DLLs

Pastikan file pendukung ikut disertakan agar aplikasi tidak error:

  • Database Driver: Jika menggunakan MySQL, sertakan libmysql.dll.
  • Database File: Jika menggunakan SQLite, sertakan file .db.
  • FastReport: Pastikan semua bpl/dll yang dibutuhkan sudah terangkum (biasanya statis di Delphi).
  • Assets: Folder gambar, icon, atau config file (.ini atau .json).

Branding & Ikon

Berikan identitas profesional pada file EXE Anda:

Project Options (Ctrl+Shift+11)

  • Ganti ikon aplikasi default (Delphi Icon) dengan ikon kustom Anda (.ico).
  • Isi Version Info: Company Name, Product Name, dan Version Number.

Software Installer

Untuk pengalaman pengguna yang lebih baik, gunakan software pembuat installer:

Inno Setup

Tool gratis dan populer untuk membuat file Setup.exe. Mendukung skrip Pascal untuk kustomisasi tingkat lanjut.

Portable Version

Cukup kompres folder aplikasi (EXE + DLL + Assets) menjadi file .zip atau .rar.
