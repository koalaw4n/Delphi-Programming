Apa itu FireDAC?

Universal Data Access

Library Delphi untuk akses data universal berkinerja tinggi ke berbagai jenis database.

Mendukung database populer:

  • MySQL / MariaDB
  • SQLite (Local)
  • PostgreSQL
  • MS SQL Server

Komponen Utama Koneksi

Ada 3 komponen dasar yang harus ada di Form/DataModule:

1. TFDConnection
Mengatur parameter koneksi (Server, User, Password).

2. TFDQuery
Tempat menulis perintah SQL (Select, Insert, Update, Delete).

3. TDataSource
Jembatan antara Query dengan komponen Visual (DBGrid).

Menampilkan Data (SELECT)

Langkah-langkah menampilkan data ke TDBGrid:

FDQuery1.Close;
FDQuery1.SQL.Text := 'SELECT * FROM tb_mahasiswa';
FDQuery1.Open;

Binding:
DBGrid.DataSource → DataSource1
DataSource1.DataSet → FDQuery1

Menambah Data (INSERT)

Cara aman menggunakan Parameters:

FDQuery1.SQL.Text := 'INSERT INTO tb_user (nama, email) VALUES (:nama, :email)';
FDQuery1.Params.ParamByName('nama').AsString := Edit1.Text;
FDQuery1.Params.ParamByName('email').AsString := Edit2.Text;
FDQuery1.ExecSQL;

ExecSQL digunakan untuk perintah yang tidak mengembalikan dataset (Insert/Update/Delete).
