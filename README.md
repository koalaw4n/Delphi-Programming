# Fundamental Programming with Delphi

Repositori ini berisi materi pembelajaran lengkap dari dasar hingga tingkat lanjut untuk menguasai pengembangan aplikasi menggunakan bahasa pemrograman Delphi (Object Pascal). Kurikulum ini disusun secara sistematis untuk membimbing Anda menjadi pengembang aplikasi desktop yang handal.

## 📋 Kurikulum Pembelajaran (Sesi 1 - 12)

Berikut adalah ringkasan materi yang dipelajari dalam kursikulum ini:

| Sesi | Topik | Deskripsi Singkat |
| :--- | :--- | :--- |
| 1 | **Pengenalan Delphi** | Instalasi IDE, konfigurasi workspace, dan pembuatan aplikasi "Hello World". |
| 2 | **Dasar Pascal** | Memahami variabel, konstanta, tipe data dasar, dan operator. |
| 3 | **Logika Percabangan** | Alur kontrol program menggunakan `if-then-else` dan `case-of`. |
| 4 | **Perulangan** | Implementasi looping dengan `for-to-do`, `while-do`, dan `repeat-until`. |
| 5 | **Modularisasi** | Teknik pembuatan Prosedur (Procedure) dan Fungsi (Function). |
| 6 | **OOP Dasar** | Konsep dasar Class, Object, Constructor, Property, dan Method. |
| 7 | **FireDAC Database** | Pengenalan koneksi database lokal menggunakan komponen FireDAC. |
| 8 | **Operasi CRUD** | Implementasi manipulasi data (Create, Read, Update, Delete). |
| 9 | **Web & REST API** | Integrasi layanan RESTful, konsumsi JSON, dan pemakaian REST Client. |
| 10 | **Multi-Threading** | Teknik asinkron agar UI tetap responsif menggunakan `TThread` dan `TTask`. |
| 11 | **Security** | Implementasi sistem Login, perlindungan data, dan hashing password (SHA-256). |
| 12 | **Deployment** | Build mode release, optimasi aplikasi, dan pembuatan installer (Inno Setup). |
