Variabel & Tipe Data

Variabel adalah wadah penyimpanan data. Delphi bersifat Strongly Typed.

Tipe Data Umum

  • Integer: Angka bulat (1, 100, -5).
  • String: Teks / Kata ("Halo").
  • Double: Angka desimal (3.14).
  • Boolean: True atau False.

Deklarasi Variabel:

var
  Nama: String;
  Umur: Integer;
begin
  Nama := 'Ikhwan';
  Umur := 25;
end;

Konversi Tipe Data

Karena Delphi ketat soal tipe data, kita sering butuh konversi:

  • IntToStr(10) → Angka ke Teks
  • StrToInt('10') → Teks ke Angka
  • FloatToStr(3.14) → Desimal ke Teks

Penting saat mengambil input dari TEdit.Text (String) untuk perhitungan matematika.

Logika Percabangan

Gunakan if untuk mengambil keputusan dalam program.

if Umur >= 17 then
begin
  ShowMessage('Boleh membuat SIM');
end
else
begin
  ShowMessage('Belum cukup umur');
end;

Ingat: Jangan beri titik koma (;) sebelum kata kunci else.

Perulangan (Looping)

FOR Loop

Digunakan saat jumlah perulangan sudah pasti.

for i := 1 to 5 do
begin
  // Aksi di sini
end;

WHILE Loop

Berulang selama kondisi masih benar.

while (Stok > 0) do
begin
  // Aksi di sini
end;
