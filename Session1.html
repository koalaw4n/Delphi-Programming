Apa itu Delphi?

Object Pascal

Bahasa pemrograman tingkat tinggi yang terstruktur, mudah dibaca, dan sangat kuat (powerful).

RAD Studio

Rapid Application Development - Memungkinkan pembuatan aplikasi Windows dalam waktu singkat secara visual.

"Delphi bukan hanya bahasa, tapi ekosistem pengembangan aplikasi yang efisien."

Persiapan Instalasi

  • Unduh Community Edition (Gratis).
  • Spesifikasi Minimum: RAM 4GB, HDD 10GB.
  • Pilih Windows Platform saat instalasi.
  • Pastikan koneksi internet stabil selama proses.

Image: Tampilan Installer Delphi

Navigasi IDE Delphi

IDE (Integrated Development Environment) Delphi terdiri dari:

1. Form Designer
Kanvas untuk mendesain UI aplikasi.
2. Object Inspector
Tempat mengatur Properti (Warna, Teks) & Event.
3. Tool Palette
Daftar komponen (Button, Edit, Label).
4. Project Manager
Mengelola file-file dalam proyek.

Membuat Proyek Pertama

Ikuti langkah berikut:

  1. Klik File > New > VCL Forms Application - Delphi.
  2. Cari TButton di Tool Palette, lalu klik dan pindahkan ke Form.
  3. Pada Object Inspector, ubah Caption tombol menjadi "Klik Saya".
  4. Klik dua kali pada tombol untuk masuk ke Code Editor.
  5. Tulis kode: ShowMessage('Halo Dunia!');
Tips: Tekan F9 untuk menjalankan aplikasi (Run).
