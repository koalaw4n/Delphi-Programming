Hash vs Enkripsi

Hashing (Satu Arah)

Data diacak dan tidak bisa dikembalikan. Digunakan untuk Password. Contoh: SHA-256.

Enkripsi (Dua Arah)

Data diacak tapi bisa dikembalikan (Dekripsi) menggunakan kunci. Contoh: AES.

Hashing di Delphi

Delphi memiliki unit System.Hash yang sangat mudah digunakan untuk keamanan dasar:

uses System.Hash;

var
  PasswordHash: string;
begin
  // Mengamankan password dengan SHA2
  PasswordHash := THashSHA2.GetHashString('rahasia123');
  ShowMessage(PasswordHash);
end;

Password yang sama akan menghasilkan Hash yang sama. Bandingkan Hash di DB dengan input user saat Login.

Langkah Login Aman

  1. Sanitize Input: Pastikan input tidak mengandung karakter berbahaya (SQL Injection).
  2. Hash Comparison: Jangan pernah bandingkan password asli. Bandingkan hasil Hash-nya.
  3. Session Management: Simpan status login (misal: ID User) di variabel global atau objek khusus selama aplikasi berjalan.

Gunakan TEdit.PasswordChar (biasanya '*') agar password tidak terlihat di layar.
